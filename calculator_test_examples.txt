===============================================================================
                         QUICK REFERENCE - FORMULAS
===============================================================================
Ohm's Law:
  V = I * R    (vir function)
  I = V / R    (ivr function)
  R = V / I    (rvi function)

Power:
  P = V * I    (pvi function)
  P = I² * R   (pir function)
  P = V² / R   (pvr function)

AC Power:
  Real Power = V * I * cos(θ)     (preal function)
  Reactive Power = V * I * sin(θ) (preact function)
  Apparent Power = V * I          (papp function)
  Power Factor = cos(θ)           (pf function)

Reactance:
  Xc = 1/(2πfC)  (xc function - capacitive)
  Xl = 2πfL      (xl function - inductive)
  Z = √(R²+X²)   (zrx function - impedance)
  f₀ = 1/(2π√LC) (fres function - resonant freq)

Decibels:
  dB(V) = 20*log10(V1/V2)  (dbv function)
  dB(P) = 10*log10(P1/P2)  (dbp function)

Voltage Divider:
  Vout = Vin * R2/(R1+R2)  (vdiv function)

Calculus:
  sum(n) = 1+2+...+n = n(n+1)/2
  sum2(n) = 1²+2²+...+n²
  sum3(n) = 1³+2³+...+n³
  geom(a,r,n) = a + ar + ar² + ... + arⁿ
  intpow(a,b,k) = ∫[a to b] x^k dx
  intexp(a,b) = ∫[a to b] e^x dx
  intsin(a,b) = ∫[a to b] sin(x) dx
  intcos(a,b) = ∫[a to b] cos(x) dx
  intlog(a,b) = ∫[a to b] 1/x dx
  derivpow(x,n,h) = d/dx(x^n) at x
  derivexp(x,h) = d/dx(e^x) at x
  derivsin(x,h) = d/dx(sin(x)) at x
  derivcos(x,h) = d/dx(cos(x)) at x
  derivln(x,h) = d/dx(ln(x)) at x
